# Screenshots

所有页面截图、设计比对图、浏览器验收图统一放在这个目录下，不再散落到仓库根目录。

## 目录约定

- `ui-iterations/`
  页面迭代截图、视觉对比图、局部改版验收图。
- `manual-smoke/`
  手工联调或冒烟验证过程中留存的截图。
- `bugs/`
  问题复现图、异常状态截图。

## 命名建议

- `YYYY-MM-DD-页面名-用途.png`
- 示例：`2026-04-08-profile-dashboard-iteration.png`

## 维护规则

- 需要保留的截图放这里。
- 临时截图在任务结束后要么归档到这里，要么删除。
- 不要再把截图直接丢到仓库根目录。
